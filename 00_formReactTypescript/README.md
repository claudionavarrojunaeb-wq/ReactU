React + TypeScript multipasos (Vite)

Instrucciones rápidas:
1. Abrir la carpeta `00_formReactTypescript`
2. Ejecutar `npm install`
3. Ejecutar `npm run dev` y abrir la URL que muestre Vite

Estructura básica creada:
- src/
  - main.tsx
  - App.tsx
  - components/MultiStepForm.tsx
  - components/steps/Step1.tsx, Step2.tsx, Step3.tsx

Si quieres que instale las dependencias aquí mismo y ejecute el dev server, dime y lo hago.
