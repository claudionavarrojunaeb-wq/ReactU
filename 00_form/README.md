Formulario multipasos simple

Archivos:
- index.html: página principal con el formulario multipasos (3 pasos + resumen)
- style.css: estilos mínimos
- script.js: lógica de navegación entre pasos y validaciones simples

Para ejecutar:
- Abre `00_form/index.html` en tu navegador local.

Notas:
- Implementación en Vanilla JS para no añadir dependencias.
- Si quieres versión React/TypeScript dime y la creo en su lugar.
