# ReactU - Curso de React + TypeScript

Curso completo de React con TypeScript creado por **Fernando Herrera** en Udemy.

## 📖 Descripción del Curso

Este curso te llevará desde los conceptos básicos de React hasta convertirte en un desarrollador experto, utilizando las mejores prácticas y las últimas características del framework. Aprenderás a construir aplicaciones modernas con React 18, TypeScript, y herramientas de desarrollo actuales.

## 🎯 ¿Qué aprenderás?

- **Fundamentos de React**: Componentes, JSX, Props, State, Eventos
- **Hooks de React**: useState, useEffect, useContext, useReducer, y hooks personalizados
- **TypeScript con React**: Tipado estático, interfaces, y mejores prácticas
- **Manejo del Estado**: Context API, Reducers, y patrones de estado
- **Rutas**: React Router para navegación entre páginas
- **Testing**: Jest y React Testing Library
- **Herramientas de Desarrollo**: Vite, ESLint, Prettier
- **APIs y HTTP**: Fetch, Axios, manejo de promesas
- **Optimización**: Lazy loading, memoización, y performance
