# Formulario multipaso — Resumen

Fecha: 2025-10-14 09:16:00 (Chile, UTC-3)

Este documento recoge lo realizado para el formulario multipaso de tres pasos en el proyecto `00-formulario`.

## Descripción

Se implementó un formulario multipaso simple con navegación entre 3 pasos, un resumen final y un envío demo (console.log + alert). Se añadió un stepper visual con círculos numerados conectados por una línea.

## Archivos creados/modificados

- `src/MultiStepForm.tsx`
  - Componente principal que controla el paso actual, el estado del formulario y provee funciones `next`, `prev`, `update` y `handleSubmit`.
- `src/steps/Step1.tsx`
  - Campo: `name` (texto).
- `src/steps/Step2.tsx`
  - Campos: `email` (email), `age` (number).
- `src/steps/Step3.tsx`
  - Campo: `agree` (checkbox) y un resumen en JSON de los datos ingresados.
- `src/index.css`
  - Estilos para el card, botones y stepper (círculos y línea conectora).
- `src/main.tsx`
  - Ahora renderiza `MultiStepForm`.

## Comportamiento

- Navegación simple con botones "Atrás" y "Siguiente". El último botón es "Enviar".
- El stepper muestra tres círculos numerados; los pasos >= current reciben la clase `active`.
- Al enviar se hace `console.log(data)` y se muestra una alerta.

## Cómo probar localmente

```powershell
cd d:\_ReactU\ReactU\00-formulario
npm install
npm run dev
```

Abrir la URL que Vite muestre (p. ej. `http://localhost:5173`).
